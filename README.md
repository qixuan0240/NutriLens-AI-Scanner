# NutriLens AI Scanner

This repository contains the key feature implementation for the NutriLens HCI project.

## Key Feature

The implemented key feature is the AI Food Scanner. It allows users to:

- Select a food image from the device
- Scan the food using a scanner animation
- View nutrition results
- Add the meal to the daily calorie dashboard
- Receive button sound feedback during interaction

## Technology Used

- Java
- Java Swing
- JFileChooser
- CardLayout
- Timer
- Java Sound API

## How to Run

1. Download the project files.
2. Make sure `Main.java`, `food.png`, and `scan.wav` are in the same folder.
3. Compile the Java file:

```bash
javac Main.java
